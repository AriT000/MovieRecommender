# MovieRecommender
Recommender for movies based on preferences

---

### How to run:  
1. Create virtual environment (venv):  
   `python3 -m venv .venv`
2. Activate virtual environment (venv):  
   `source .venv/bin/activate`
3. Install Bottle:  
   `pip install bottle`
4. Run server on http://127.0.0.1:8080/:  
   `cd MovieRecommender`
   `python3 main.py`
